## Assignment 1 — Python, IPython, Packages, and Jupyter Notebooks

The first lesson is to follow along with the [Jupyter notebooks](../lessons/lesson01.ipynb).

Get comfortable running Jupyter notebooks in your system and make sure that you understand the basics:

1) Download the repository according to the instructions
2) Activate your environment and launch the Jupyter notebook on the current folder. 
3) Running individual cells – do they all run? 
4) Navigating the notebook interface – can you use only your keyboard to get around?
5) Add a Markdown cell at the end of the Notebook with the content `### Exercises`.
   **Hint**: you can change the cell type at the top menu.
6) Add a new cell at the end and write some code to it: try to loop through the numbers from 1 to 100 
   and print them.
7) Add a new code cell to make the code instead go from 100 to 1.
   **Hint**: look at the `range` function with a step argument.
8) Installing missing packages – try to install the package 
   [`pandas`](https://pandas.pydata.org/docs/getting_started/install.html) by yourself in one of the cells. Import the
   package using `import pandas as pd`.
